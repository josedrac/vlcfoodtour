 Contributors: DavidoffNeal
 Requires at least: 3.9
 Tested up to: 4.3.1
Stable tag: 1.0.1

  == Description ==
  One of the most advanced WordPress follow button plugins available.

  == Installation ==  
* Upload the 'simple-follow-buttons-plus' folder to the /wp-content/plugins/ directory.
 * Activate the plugin through the 'Plugins' menu in WordPress. 
* Navigate to the Simple Follow Buttons Plus Admin Panel via ‘Follow Buttons’ to change your settings as desired. 
* Save then activate your licence via the 'Licence' page.
